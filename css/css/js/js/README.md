# ZewedJobs - Ethiopia's #1 Job Platform

A modern, responsive job platform for Ethiopian job seekers and employers.

## Features

- **Job Search**: Advanced filtering by location, experience, salary, and date
- **Free Courses**: Access to career-enhancing training programs
- **Business Directory**: Featured Ethiopian businesses
- **Social Media Integration**: Telegram, Facebook, LinkedIn, YouTube
- **Mobile-Friendly**: Fully responsive design
- **PWA Support**: Installable as a mobile app
- **Ad System**: Google AdSense integration with fallback ads
- **Payment Integration**: Support for Ethiopian payment methods (Telebirr, CBE, HelloCash, Amole)
